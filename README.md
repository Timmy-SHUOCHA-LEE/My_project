# My_project
(1) ETF專案 : 與市場比對，看第一金需要新增與剔除的ETF標的。
(2) 0050與0051成分股專案 : 利用每月0050、0051的成分股去比對第一金0050、0051的成分股需要剔除與新增的標的。 
(3) 集中市場相關數據專案 : 利用關鍵字查詢、pdf檔的轉換及行列的抓取方式抓出集中市場的部分數據數據來供公司的同仁使用
(4) 月與年的工作表 : 抓取全部有關月與年的上市櫃的證券成交統計與三大法人買賣超分別放進月與年一個Excel當中，用日月年來當作分隔，最後做出dashboard的方式去呈現數據。
(5) PTP商品專案 : 抓取一個PTP商品的總集合商品標的與利用排程每月需要新增的標的，以及含有寄信的程式碼在裡面，以供公司的同仁使用。
(6) 投資組合 : 工作後的興趣是自己鑽研財金相關的知識並撰寫程式，從籌碼面與績效等等的條件去了解個股與加權指數比較後的表現。
